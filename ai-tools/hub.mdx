## Como funciona?

Siga os passos abaixo para começar a documentar com IA:

<Steps>
  <Step title="Baixe o OpenAPI">
    Faça o download do nosso arquivo `openapi.json` na seção de Referência da API.
  </Step>
  <Step title="Upload para a IA">
    Envie o arquivo para o chat da sua IA escolhida e peça para ela analisar os endpoints.
  </Step>
  <Step title="Gere seu Código">
    Solicite exemplos de integração: *"Gere um código em Node.js para criar um pagamento PIX usando a API da Sync Pay"*.
  </Step>
</Steps>
